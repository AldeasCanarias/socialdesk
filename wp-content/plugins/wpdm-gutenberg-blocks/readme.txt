Gutenberg editor blocks for WordPress Download Manager


== Description ==

Gutenberg Blocks for WordPress Download Manager is the collection of blocks for the new Gutenberg block editor to use various download manager features like embedding package or category within a page/post. However, we also will be adding all variations of UI blocks so that only this block plugins becomes enough for your need.

= WPDM Blocks =
* Package - a block to show a download/file/package
* Packages - query download managers items using various filters, like `WP_Query` for download manager items
* Category - a block to list downloads from the selected category/categories
* Category Blocks - shows selected download manager categories in UI blocks with different styles


= UI Blocks =
* Panel
* Call To Action
* Slider

== Installation ==

1. Upload `wpdm-gutenberg-blocks` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
